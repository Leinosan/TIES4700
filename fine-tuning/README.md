# Fine tuning example

You need to add the RODI-DATA to the fine-tuning folder.

To split the training data 80/20 run:

`pip install scikit-learn`

`python split_script.py`

There are two fine tuning examples, a one using Resnet18 and one Resnet50 located in files with respective names. You can run the fine tuning:

`python install matplotlib`

`python fine_tuning_resnet50.py`

The model with better results is saved here: https://drive.google.com/file/d/1mvPT-AzAn_H_ZJkInDLitl2wKU-wNWAS/view?usp=drive_link
